# Lunaerem's Personal Website
A [Frutiger Aero](https://en.wikipedia.org/wiki/Frutiger_Aero)-inspired personal website for myself that acts as a portfolio and place where I can upload and showcase some of my projects.

Image here

## Usage
The live website can be found [here](https://lunaerem.github.io/personal-website/).
> [!NOTE]
> The website is not necessarily intended to be cloned and run locally, since it contains a lot of bespoke code designed for my particular use case. However, if anyone wishes to look over the source code for inspiration or examples, they are free to do so.

## Technologies & Attributions
- Made using CSS, HTML, TypeScript, [Vite](https://vite.dev/), and [Vue](https://vuejs.org/)
- Hosted using GitHub Pages
- Routing powered by [Vue Router](https://router.vuejs.org/)
- Build/Dev using NPM
- Using the [Open Sans](https://fonts.google.com/specimen/Open+Sans) font
